# React + ES6 + GA Reporting API example 

### Purpose
Simple project to show how to use the gapi javascript lib with react to get google analytics data.

After you log in, you'll be able to click on a GA profile and get a console.log() of the API response for the numbers of sessions for the last 7 days.

### Tech
- ES6 (Babel),
- react,
- react-bootstrap,
- eslint,
- npm,
- webpack,
- gapi lib.

### How-to

```
npm install
npm start
Open http://localhost:5000
```

### Linting

ESLint with React linting options have been enabled.

```
npm run lint
```
